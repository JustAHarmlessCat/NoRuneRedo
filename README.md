**Mod Overview:**

This Mod takes a look if there are Runes already applied to the Weapon you are trying to enchant.
If there are Runes already applied to the Weapon the Mod will prevent the Game from Opening the Spell Crafting Menu.
This adds consequence to your Decisions and makes finding new Weapons more important than in the Base Game.

**How to install:**

1. Install BepInEx 6.0 (IL2CPP)
2. Paste the .dll into the Bepinex/plugins folder
